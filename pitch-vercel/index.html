Technical Foundation

100% Stellar-native

Auth

SEP-10

Standard Stellar authentication. No workarounds. Compatible with existing accounts.

Recovery

SEP-30

Non-custodial account recovery. Multi-factor identity verification. Signer revocation.

Key Mgmt

SSS + TEE

2-of-3 Shamir Secret Sharing. Ephemeral keys inside Trusted Execution Environments.

User never touches a private key but never loses control.
