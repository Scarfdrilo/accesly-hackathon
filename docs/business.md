# 💼 Modelo de Negocio - Accesly

## Producto: Dos Capas

### Capa 1: Para Usuarios
- Login con Google → wallet instantánea
- Agente AI con wallet propia
- Zero setup, zero code
- URL: accesly.com

### Capa 2: Para Developers
- SDK npm: `npm install accesly`
- 3 líneas de código para integrar
- API completa
- URL: accesly.com/developers
